# 项目总结

## 项目概述

**Ziwei Web** 是一个基于 React + Vite + Tailwind CSS 的紫微斗数专业排盘软件 Web UI，提供交互性强、信息直观的命盘可视化和分析功能。

## 已实现功能

### ✅ 核心功能
- [x] 出生信息输入表单（性别、日期、时间、时辰）
- [x] 一键排盘，调用 `@ziwei/core` 引擎
- [x] 12 宫格 SVG 命盘可视化
- [x] 交互式选择系统（单选/多选）
- [x] 推导日志查看器
- [x] 图层系统（6 种图层 + 4 种预设）
- [x] 星曜搜索功能

- [x] 规则集切换

## 技术架构

### 前端技术栈
- **React 19** - UI 框架
- **TypeScript** - 类型安全
- **Vite 7** - 构建工具
- **Tailwind CSS 4** - 样式框架
- **Zustand 5** - 状态管理

### 核心引擎
- **@ziwei/core** - 紫微斗数核心引擎（workspace 依赖）

### 组件结构
```
src/
├── components/
│   ├── layout/          # 布局组件（3 个）
│   ├── chart/           # 命盘组件（4 个）
│   ├── input/           # 输入组件（3 个）
│   ├── inspector/       # 检查器组件（4 个）
│   └── timeline/        # 时间轴组件（1 个）
├── stores/              # 状态管理（1 个）
├── types/               # 类型定义（1 个）
└── ...
```

### 状态管理
使用 Zustand 管理全局状态：
- 命盘数据（chart、trace、lastBirth）
- UI 状态（selection、layers、timeline）
- 交互状态（search、hover、compare）

## 如何运行

### 1. 安装依赖
```bash
cd D:\Ziwei\packages\web
pnpm install
```

### 2. 启动开发服务器
```bash
pnpm dev
```

### 3. 访问应用
打开浏览器访问 http://localhost:5173

### 4. 开始使用
1. 在左侧输入出生信息
2. 点击"排盘"按钮
3. 查看中间的命盘
4. 点击宫位/星曜查看详情
5. 使用图层控制调整显示内容
