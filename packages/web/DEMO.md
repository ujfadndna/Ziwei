## 核心功能演示

### 1. 排盘流程

#### 步骤 1：输入出生信息
```
性别：男
日期：2000-01-01
时间：08:00
时辰：辰 (07-08) [自动]
```

#### 步骤 2：点击"排盘"按钮
系统调用 `@ziwei/core` 引擎生成命盘。

#### 步骤 3：查看命盘
中间画布显示 12 宫格命盘：

```

### 2. 交互操作

#### 选择宫位
**操作**：点击"命宫"
**效果**：
- 宫位高亮（蓝色边框）
- 右侧显示宫位详情：
  ```
  命宫 (卯)
  ─────────────
  地支：卯
  天干：甲
  纳音：大溪水

  星曜列表：
  • 贪狼 (主星) 化科
  • 左辅 (辅星)

  三方四正：
  • 对宫：迁移宫 (酉)
  • 三合：官禄宫 (申)、财帛宫 (子)
  ```

#### 选择星曜
**操作**：点击"紫微"星
**效果**：
- 星曜高亮（黄色光环）
- 右侧显示星曜详情：
  ```
  紫微
  ─────────────
  类型：主星
  亮度：庙
  四化：化禄（庚干）

  所在宫位：父母宫 (巳)

  推导日志：
  1. 根据出生年干（庚）定紫微位置
  2. 紫微在巳宫
  3. 庚干四化：紫微化禄
  ```

#### 多选
**操作**：Shift+点击多个宫位/星曜
**效果**：同时选中多个项目，右侧显示第一个选中项的详情。

#### 清空选择
**操作**：按 ESC 或点击"清空选择"按钮
**效果**：清空所有选中状态。

### 3. 搜索功能

#### 搜索星曜
**操作**：在右侧搜索框输入"紫微"
**效果**：
- 自动定位到"紫微"所在宫位（父母宫）
- 宫位和星曜高亮
- 显示搜索结果：
  ```
  找到：紫微 (父母宫)
  [点击选中]
  ```

#### 模糊搜索
**操作**：输入"天"
**效果**：
- 显示第一个匹配的星曜（如"天机"）
- 支持部分匹配

### 4. 图层控制

#### 预设模式切换

**入门模式**：
```
显示：主星 + 四化 + 关系线
隐藏：辅星、煞曜、大限

适合：初学者快速了解命盘结构
```

**专业模式**（默认）：
```
显示：全部图层
适合：专业命理师深度分析
```

**极简模式**：
```
显示：仅主星
隐藏：所有其他内容

适合：快速查看主星分布
```

**四化专注**：
```
显示：主星 + 四化
突出：四化信息

适合：专注四化分析
```

#### 自定义图层

**开启/关闭单个图层**：
- ☑ 主星
- ☑ 辅星 / 杂曜
- ☐ 煞曜（擎羊/陀罗/火铃/空劫）
- ☑ 四化
- ☑ 三方四正关系线
- ☐ 运限（大限等）

**效果**：实时更新命盘显示内容。

### 5. 关系线显示

#### 三方四正关系
**操作**：开启"关系线"图层，点击"命宫"
**效果**：
```
         午
          │
          │
卯 ───────┼─────── 酉
(命宫)    │      (迁移宫)
          │
         子
```

显示：
- 对宫连线（命宫 ↔ 迁移宫）
- 三合连线（命宫 ↔ 官禄宫 ↔ 财帛宫）

### 6. 推导日志

#### 开启推导日志
**操作**：勾选"推导日志"选项后排盘
**效果**：右侧显示详细推导过程：

```

─────────────────────

• 输入：出生时辰 = 辰时 (4)

步骤 155: 四化 - 武曲化权

### 7. 深色模式

#### 切换主题
**操作**：点击右上角"深色/浅色"按钮
**效果**：
- 浅色模式：白色背景，黑色文字
- 深色模式：深灰背景，浅色文字
- 自动保存偏好设置

### 8. 规则集切换

#### 选择规则集
**操作**：在左侧"规则集"下拉菜单选择
**选项**：
- 默认规则
- 中州派
- 三合派
- 飞星派

**效果**：使用不同规则集重新排盘，星曜位置可能不同。

### 9. 时间轴（未来功能）

#### 切换时间模式
**操作**：点击底部时间轴按钮
**模式**：
- 本命：显示本命盘
- 大限：显示大限盘（叠加或替换）
- 流年：显示流年盘
- 流月：显示流月盘

**效果**：根据选择的时间模式更新命盘显示。

## 颜色说明

### 宫位标记
- 🟢 绿色"命"标签：命宫
- 🟣 紫色"身"标签：身宫
- 🔵 蓝色边框：选中的宫位
- ⚪ 灰色背景：Hover 的宫位

### 星曜颜色
- 🟡 黄色：主星（紫微、天机、太阳等）
- 🔵 蓝色：辅星（文昌、文曲、左辅、右弼等）
- 🔴 红色：煞曜（擎羊、陀罗、火星、铃星等）
- ⚪ 灰色：其他星曜

- 🟢 绿色"禄"：化禄

## 使用技巧

### 1. 快速定位星曜
使用搜索功能快速找到目标星曜，无需手动查找。

### 2. 对比不同规则集
切换规则集后重新排盘，对比不同派别的差异。

### 3. 学习推导过程
开启推导日志，了解每个星曜的安星规则和四化逻辑。

### 4. 简化视图
使用"极简"预设，专注于主星分布，减少干扰。

### 5. 多选分析
Shift+点击多个星曜，批量查看详情。

## 常见问题

### Q: 为什么有些星曜不显示？
A: 检查左侧图层开关，确保对应图层已开启。

### Q: 如何查看三方四正关系？
A: 开启"关系线"图层，点击宫位后会显示连线。

### Q: 推导日志太长，如何快速查找？
A: 使用浏览器的查找功能（Ctrl+F）搜索关键词。

### Q: 如何保存命盘？
A: 目前支持截图保存，未来将支持导出 PDF/图片。
