# 快速启动指南

## 一键启动

```bash
cd D:\Ziwei\packages\web
pnpm dev
```

访问 http://localhost:5173

## 使用流程

### 1. 输入出生信息
在左侧控制台填写：
- **性别**：男/女
- **日期**：公历日期（例如：2000-01-01）
- **时间**：24小时制（例如：08:00）
- **时辰**：自动根据时间推算，也可手动选择

### 2. 排盘
点击"排盘"按钮，系统将生成命盘。

### 3. 查看命盘
- 中间画布显示 12 宫格命盘
- 每个宫位显示：宫名、地支、天干、星曜列表
- 命宫标记为绿色"命"标签
- 身宫标记为紫色"身"标签

### 4. 交互操作

#### 选择宫位/星曜
- **单击宫位**：选中宫位，右侧显示详情
- **单击星曜**：选中星曜，右侧显示详情
- **Shift+单击**：多选宫位/星曜
- **ESC**：清空所有选择

#### 搜索星曜
1. 在右侧搜索框输入星曜名称（例如："紫微"）
2. 系统自动定位并高亮该星曜所在宫位
3. 点击搜索结果自动选中

### 5. 图层控制

在左侧控制台切换图层：

#### 预设模式
- **入门模式**：主星 + 四化（适合初学者）
- **专业模式**：全部显示（默认）
- **纯净模式**：仅主星
- **四化专注**：主星 + 四化

#### 自定义图层
- **主星**：紫微、天机、太阳等 14 主星
- **辅星**：文昌、文曲、左辅、右弼等
- **煞曜**：擎羊、陀罗、火星、铃星等
- **四化**：禄、权、科、忌
- **关系线**：三方四正连线
- **大限**：大限宫位标记

### 6. 推导日志

勾选"推导日志"选项后排盘，可在右侧检查器查看详细的推导过程：
- 每个星曜的安星规则
- 四化的推导逻辑
- 宫位的计算过程

用于学习紫微斗数原理和调试。

## 快捷键

- **ESC**：清空选择
- **Shift+Click**：多选

## 主题切换

点击右上角"深色/浅色"按钮切换主题。

## 常见问题

### Q: 排盘后没有显示命盘？
A: 检查浏览器控制台是否有错误信息，确保 @ziwei/core 引擎正常工作。

### Q: 星曜显示不全？
A: 检查左侧图层开关，确保对应图层已开启。

### Q: 如何查看某个星曜的详细信息？
A: 点击星曜标签，右侧检查器会显示详细信息和推导日志。

### Q: 如何查看三方四正关系？
A: 开启"关系线"图层，点击宫位后会显示三方四正的连线。
